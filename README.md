-- Project Overview --

This project analyzes Vrinda Store sales data using Microsoft Excel. The dashboard provides interactive insights into sales performance, customer demographics, order trends, and sales channels using Pivot Tables, Pivot Charts, Slicers, and KPIs.

-- Objective --

The objective of this project is to help business owners understand:

Sales performance
Customer buying behavior
Best-selling products
Monthly sales trend
Order status
Top performing states
Sales channels contribution
🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning
Data Validation
 

* The dataset contains Vrinda Store sales information including:

Order ID
Customer

Age
Category
Quantity
Sales
State
Month
Channel
Order Status

✔ Total Sales

✔ Interactive Slicers

 ---Business Insights---
1 Women customers generated more sales than men.
2️ Amazon contributed the highest number of orders among all sales channels.
3️ Maharashtra was the highest revenue-generating state.
4️ Adult customers placed the maximum number of orders.
5️ March recorded the highest monthly sales.
6️ Most orders were successfully delivered.
